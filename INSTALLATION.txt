📋 الخطوة 1: إعداد Firebase
═══════════════════════════════════════════════════════════════════

1️⃣ إنشاء مشروع Firebase:
   ⭐ اذهب إلى: https://console.firebase.google.com
   ⭐ اضغط "Add project" أو "إضافة مشروع"
   ⭐ أدخل اسم المشروع (مثلاً: GmailChatApp)
   ⭐ اتبع الخطوات حتى إنشاء المشروع

2️⃣ تفعيل Realtime Database:
   ⭐ في القائمة الجانبية، اختر "Realtime Database"
   ⭐ اضغط "Create Database"
   ⭐ اختر الموقع (مثلاً: us-central1)
   ⭐ اختر "Start in test mode" للبداية

3️⃣ إعداد قواعد Database:
   ⭐ اضغط على تبويب "Rules"
   ⭐ استبدل القواعد بالتالي:

   {
     "rules": {
       ".read": true,
       ".write": true
     }
   }

   ⭐ اضغط "Publish"

4️⃣ الحصول على بيانات Firebase:
   ⭐ اذهب إلى Project Settings (أيقونة الترس)
   ⭐ اختر "Your apps"
   ⭐ اضغط على أيقونة "</>" (Web)
   ⭐ سجل التطبيق
   ⭐ انسخ بيانات firebaseConfig

5️⃣ تحديث الملف:
   ⭐ افتح ملف index.html
   ⭐ ابحث عن "const firebaseConfig"
   ⭐ استبدل القيم بقيمك من Firebase:

   const firebaseConfig = {
       apiKey: "ضع_هنا_مفتاح_API",
       authDomain: "ضع_هنا_AUTH_DOMAIN",
       databaseURL: "ضع_هنا_DATABASE_URL",
       projectId: "ضع_هنا_PROJECT_ID",
       storageBucket: "ضع_هنا_STORAGE_BUCKET",
       messagingSenderId: "ضع_هنا_SENDER_ID",
       appId: "ضع_هنا_APP_ID"
   };

📋 الخطوة 2: الرفع على GitHub Pages
═══════════════════════════════════════════════════════════════════

1️⃣ إنشاء Repository:

   ⭐ لا تختر "Add README"
   ⭐ اضغط "Create repository"

2️⃣ رفع الملفات:

   طريقة 1 - باستخدام GitHub Website:
   ────────────────────────────────────────
   ⭐ اضغط "uploading an existing file"
   ⭐ اسحب جميع الملفات وأفلتها
   ⭐ اضغط "Commit changes"

   طريقة 2 - باستخدام Git Commands:
   ────────────────────────────────────────
   # في مجلد المشروع، افتح Terminal واكتب:
   
   git init
   git add .
   git commit -m "Initial commit - Gmail Chat v4"
   git branch -M main
   git remote add origin https://github.com/اسمك/Gmail-Chat.git
   git push -u origin main

3️⃣ تفعيل GitHub Pages:
   ⭐ اذهب إلى Settings في المستودع
   ⭐ اختر "Pages" من القائمة الجانبية

   ⭐ اضغط "Save"
   ⭐ انتظر دقيقة ثم ستجد الرابط في الأعلى
